# Shiny App Template

Template for a basic shiny dashboard.

Features include :
  - Reactivity (values and ui)
  - CSS style sheet
  - Code segmentation
  
  
***
  
  
### 1) Installation

```
install.packages(c("shiny", "shinydashboard", "shinyFiles", "shinyWidgets", "shinyjs", "shinybusy", "shinyBS")
```
